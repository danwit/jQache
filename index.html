jQache by danwit

A simple jQuery selector (object) cache

jQache is a simple jQuery selector (object) cache. Simply add ".q" after "$" when selecting elements and you should be good to go. Don't wrap jQuery again and again and again... on DOM-elements. Use a selector cache in combination with carefully chaining.

Dependencies

jQuery

Install

Just load after jQuery.
